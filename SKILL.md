# blog-interpretation

深度解读热门内容，提取结构化方法论。

## 核心能力

- 将任意长视频/播客/博客内容转译为结构清晰的提炼报告
- 分析内容的叙事结构、表达技巧、观点框架
- 输出可直接用于内容创作的公式和模板
- 识别内容的传播要素和受众共鸣点

## 依赖工具

| 工具 | 用途 |
|---|---|
| `opencli-rs youtube transcript` | 获取YouTube视频字幕 |
| `exec` | 运行 opencli 命令 |
| `feishu_create_doc` | 将解读结果推送至飞书云文档 |

## 执行流程

### 第一步：获取内容

获取视频/播客的完整字幕或文字稿：

```bash
opencli-rs youtube transcript "[VIDEO_ID]" --format md
```

### 第二步：结构分析

对照 `references/content-analysis-framework.md` 进行逐项分析：

1. **开场结构**：悬念/反差设置、权威背书、信息量承诺
2. **展开结构**：三层递进（概念→理论→实例）、正反合对比
3. **表达技巧**：类比翻译、数据展示反差、节奏控制
4. **结尾结构**：金句收尾、总结压缩（4法则/4象限）
5. **选题规律**：权威人物、颠覆认知、时效性强、实用价值

### 第三步：提炼公式

从分析结果中提取：

- 可套用的**框架模板**（3种类型）
- 可直接引用的**金句库**
- 内容生产的**检查清单**
- 专属**词汇转化表**（专业→通俗）

### 第四步：输出

按以下格式整理，通过 `feishu_create_doc` 推送：

```
# 内容解读报告：[来源标题]

## 基本信息
来源、人物、时长、核心观点（一句话）

## 开场结构分析
[分析结果]

## 内容展开逻辑
[结构图谱/要点]

## 核心方法提炼
[可复用的框架/公式]

## 金句库
[摘录]

## 传播要素分析
[为什么能火]

## 适用场景
[哪些创作可直接套用]
```

---

## 核心原则

- **不只总结，要解构**：揭示内容为什么有效，而非仅提炼要点
- **可操作，可复用**：每个分析都要产出可直接落地的创作公式
- **翻译为易懂语言**：把专业概念转化为创作者能用的白话表述
